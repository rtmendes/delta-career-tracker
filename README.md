# Delta Career Tracker

Delta Air Lines career pipeline tracker — React app

## Tech Stack

- HTML/CSS/JavaScript

## Getting Started

```bash
git clone https://github.com/rtmendes/delta-career-tracker.git
cd delta-career-tracker
```

Open `index.html` in your browser to view the project.

## Deployment

This project is configured for deployment on [Vercel](https://vercel.com).
